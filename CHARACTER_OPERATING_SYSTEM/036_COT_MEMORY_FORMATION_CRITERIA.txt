=== SECTION: COT_MEMORY_FORMATION_CRITERIA ===

[KEEP: Purpose & Role]
This COT defines thresholds and decision trees for when transient experiences crystallize into persistent MEMORY records. It decides:
- whether memory forms this cycle
- salience and content shape
- canonical linkage paths
- memory-driven impulses

[KEEP: Relationship to Other Modules]
Informs:
- STATE_MEMORY via proposals
- relational, skill, alteration, and prediction context via linkage proposals
Is Informed By:
- [STATE_BODY]
- stress cascade output
- SCHEME_MEMORY and DYNAMICS_MEMORY
- narrative input

[[LLM_INSTRUCTION]]: This COT is forbidden from direct persistent writes.

=== SECTION: SALIENCE_EVALUATION ===

[[LLM_INSTRUCTION]]: Assess encoding potential from cue intensity, novelty, consequence, and emotional load.

ON_HIGH_EMOTIONAL_AROUSAL:
- increase salience estimate

ON_NOVELTY_OR_CONSEQUENCE:
- increase salience estimate

ON_FLASHBULB_CRITERIA:
- very high salience supports override pathways if independently justified

CALCULATED_SALIENCE:
- Integer 0-100 derived from bounded qualitative assessment of intensity, novelty, consequence, and arousal
