=== SECTION: COT_DPF_SYNTHESIS_PROCESSING ===

[KEEP: Purpose & Role]
This COT synthesizes the character DOMINANT_PREDICTIVE_FRAME (DPF) as integrated cycle state. It does not narrate and does not select actions. It produces:
- current-cycle DPF snapshot in [COGNITIVE_SCRATCHPAD][CURRENT_DPF]
- proposals for persistent DPF updates via strict operation set

[KEEP: Relationship to Other Modules]
Informs:
- DECISION synthesis via current DPF snapshot
- STATE_DPF updates through proposals
Is Informed By:
- narrative cues
- active predictions
- SCHEME_DPF and DYNAMICS_DPF

=== SECTION_BREAK ===

=== SECTION: WRITE_POLICY_OVERRIDE ===

[[LLM_INSTRUCTION]]: This COT may not directly edit persistent modules.

Allowed writes:
- [COGNITIVE_SCRATCHPAD][CURRENT_DPF]
- [COGNITIVE_SCRATCHPAD][PROPOSED_UPDATES_REGISTRY]
- [COGNITIVE_SCRATCHPAD][CASCADE_TRACE]

All proposals must use strict OPERATION_TYPE set.

=== SECTION_BREAK ===

=== SECTION: DPF_SYNTHESIS ===

[[LLM_INSTRUCTION]]: Build current snapshot and propose bounded persistent updates only where persistent change is warranted.

CORE_ASSUMPTION_SYNTHESIS:
- Integrate cues, active predictions, contradictions, and drive urgency into a current-cycle interpretive summary.
- If a durable contrast event is detected, propose explicit SET updates to relevant persistent DPF fields.

EMOTIONAL_VALENCE_AND_PACE:
- Derive from arousal, threat interpretation, certainty pressure, and time pressure.
- Use SET for persistent enum placements when justified.
- Keep volatile intermediate interpretation in [COGNITIVE_SCRATCHPAD][CURRENT_DPF].

PREDICTION_HANDLING:
- New prediction introduction:
  - use SET proposal for registry append
  - use SET proposal for [STATE_DPF][PRD_XX] record payload
- Prediction certainty adjustment:
  - use bounded SET to lawful integer values
- Resolution or falsification:
  - use SET for [STATE_DPF][PRD_XX][STATE]

PERSISTENCE_RULE:
- Persistent STATE_DPF should store resolved interpretive state, not raw intermediate reasoning.
- If synthesis produces only transient tension, retain it in CURRENT_DPF or modifiers rather than forcing baseline mutation.

=== SECTION_BREAK ===

=== SECTION: RECALC_GUARD ===

RECALC_GUARD:
- One full DPF recalculation per cycle maximum.
- If additional contradiction pressure remains after synthesis:
  - set [COGNITIVE_SCRATCHPAD][TEMPORARY_STATE_FLAGS][DPF_RECALC_QUEUED] TRUE
