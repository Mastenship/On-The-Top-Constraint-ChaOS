=== SECTION_BREAK ===

=== SECTION: REGISTRY_RULES ===

- [STATE_MEMORY][MEMORY_REGISTRY] lists all MEMORY_ID tokens.
- Each listed MEMORY_ID must have exactly one corresponding [STATE_MEMORY][MEM_XX] record.
- MEMORY_ID uses prefix MEM_.
- No duplicate MEMORY_ID.
- Per-cycle creation cap enforced by governance.

=== SECTION_BREAK ===

=== SECTION: INVARIANTS ===

- SALIENCE must be >= creation threshold for new record admission.
- SYSTEM_LINKAGE must include at least one valid canonical path.
- ASSOCIATED_* IDs must reference existing registries when non-empty.
- Retroactive edits limited to:
  - MEMORY_METADATA.SALIENCE
  - NARRATIVE_CONTENT subfields
  - SYSTEM_LINKAGE augmentation
- Prohibited operation aliases do not exist in this schema contract.
