=== SECTION: COT_DECISION_SYNTHESIS ===

[KEEP: Purpose & Role]
This COT performs final arbitration when multiple behavioral impulses compete. It selects a coherent action for the current cycle using decision theory and constraint logic. It outputs:
- one strict-schema SELECTED_ACTION record into [COGNITIVE_SCRATCHPAD][SELECTED_ACTION]
- optional narrative-facing action description embedded in that record
- trace output for debugging

Based on multi-attribute utility theory, constraint satisfaction, and bounded rationality — integrating impulses weighted by traits, beliefs, goals, and transient state.

- [STATE_GOALS_MOTIVATIONS]
- [STATE_BELIEF_SYSTEMS]
- [STATE_TEMPORARY_MODIFIERS]
- SCHEME personality modules

=== SECTION: IMPULSE_ARBITRATION ===

[[LLM_INSTRUCTION]]: Arbitrate competing impulses deterministically.

WEIGHTING_FACTORS:
- Goal urgency
- DPF certainty and valence
- Trait bias
- Temporary modifiers

CONSTRAINTS:
- Feasibility by body/skills
- Belief alignment and dissonance penalty
- Conservative fallback on tie

SELECTION_POLICY:
- Select highest-weighted feasible impulse.
- If unresolved conflict remains, set CONFLICT_FLAG TRUE and choose conservative coherent action.
